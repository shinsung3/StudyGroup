# 모든 기능 설명

### 로그인

- 흐름: 로그인 클릭
  - [x] 아이디 및 비밀번호 작성
  - [x] 아이디저장선택가능
  - [x] 회원,비회원(로그인안되어있는경우),관리자판단 후 권한 부여
  - [x] 회원은 로그인 완료시 상단메뉴에 MY Group메뉴가 생기고 login 글씨가  사람모양 아이콘으로 바뀜. 
  - [x] 회원, 비회원, 관리자 구분. 
  - [x] 아이디, 비밀번호를 이용하여 로그인 한다. 
  - [x] 회원가입이 되어있는 아이디인데 비밀번호 틀렸을경우 비밀번호틀렸다는 빨간문구가 비밀번호 밑에 표시됨.
    - [x] ->아이디가 틀렸을경우 없는 아이디라는 빨간문구가 아이디 밑에 표시됨. 
    - [x] ->비밀번호를 까먹었을 경우, 비밀번호를 잊으셨나요? 클릭 후 비밀번호 초기화 이메일을 입력하여 전송된 이메일에서 새로운비밀번호입력하여 변경.  
    - [x] 사전에 회원가입이 되어있으면 로그인 된다. 
    - [ ] 회원은 읽고, 쓰기,스터디즐겨찾기, 내 스터디목록보기,스터디 생성하기 권한, 비회원은 읽기 권한만 부여, 관리자는 읽고,쓰기,스터디즐겨찾기,내스터디목록보기 및 모든 게시글 수정,이동,삭제,오픈채팅 링크보기 권한 부여 
    - [x] 입력이 안된 빈칸이 있을 시 '이 입력란을 작성하세요' 표시 

### 회원가입

- 흐름: 사용자가 로그인 화면에서 회원가입 버튼을 눌렀을 경우 회원가입 페이지로 전환 
- 기능: 
  - [x] 메일 인증 절차 필요, [메일주소], [아이디],[닉네임],[비밀번호], [비밀번호확인]을 기입 받음 
  - [x] 입력이 안된 빈칸이 있을 시 '이 입력란을 작성하세요' 표시 
  - [x] 이미 존재하는 닉네임일 경우 생성불가 ' 이미 존재하는 닉네임입니다' 빨간문구 표시 
  - [ ] 회원가입 버튼 클릭시 이메일로 인증url 전송 -> 사용자가 이메일확인 후 클릭시 인증완료 
  - [x] 회원가입 완료 
  - [x] 관리자 권한을 가진 아이디가 필요함.
  - [ ] 이메일 회원가입시 입력된 메일주소로 인증 url 발송

### My Group

- 참여중인 그룹
  - 흐름: 로그인한 회원에게만 My group 버튼이 상단 탭에 표시됨. 회원이 모집중인 스터디에서 참여하기 버튼을 클릭한 후 해당 스터디가 모집완료(작성자가 임의로 완료 or 모집기간이 끝나면 자동으로 완료됨)되었을 시점에 my group에 해당 스터디 목록이 생성됨. 
    1. [참여중],[참여신청중], [찜한스터디],[참여완료] 을 선택해서 해당 스터디 목록을 볼 수 있음. 선택된 버튼은 글씨가 빨간색으로 변함 
    2. [참여중] 클릭 시 [닉네임님이 참여중인 스터디] 로 글씨가 써있고 우측3번째사진과 같이 [study]화면의 스터디 목록과 똑같은 형식,틀으로 스터디 목록이 보여짐. 
    3. [참여신청중] 클릭 시 [닉네임님이 참여신청중인 스터디로 글씨가 써있고 우측3번째사진과 같이 [study]화면의 스터디 목록과 똑같은 형식,틀으로 스터디 목록이 보여짐. 3.[찜한스터디] 클릭 시 [닉네임님이 찜한 스터디]로 글씨가 써있고 우측3번째사진과 같이 [study]화면의 스터디 목록과 똑같은 형식,틀으로 스터디 목록이 보여짐. 
    4. [참여완료] 클릭 시 [닉네임님이 참여완료한 스터디]로 글씨가 써있고 우측3번째사진과 같이 [study]화면의 스터디 목록과 똑같은 형식,틀으로 스터디 목록이 보여짐. -> 스터디 생성시에 설정된 스터디기간에 도달하면 참여중인 스터디에서 참여완료 스터디로 이동됨. 5. 스터디 목록의 내용및 표기는 [메인화면]의 6번에서 [모집중],[모집완료] 부분만 제외하고 동일
